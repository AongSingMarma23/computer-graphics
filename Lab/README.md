# Computer Graphics Lab (OpenGL)

This project contains basic **Computer Graphics programs** implemented using **OpenGL (GLUT)** in C/C++.

## Programs Included
- Mid Point Circle Algorithm  
- DDA Line Drawing Algorithm  
- Bresenham Line Drawing Algorithm  
- 8Ã—8 Chess Board Design  
- Flag of Bangladesh (Green rectangle with red circle)

## Tools Used
- Language: C / C++
- Graphics Library: OpenGL (GLUT)
- IDE: Code::Blocks
- Compiler: MinGW

## How to Run
- Open the project in Code::Blocks
- Link libraries: `glut32`, `opengl32`, `glu32`
- Build and run the program

## Output
- Displays lines, circles, chessboard, or flag in an OpenGL window

## Note
Only one `main()` function should be active at a time.
